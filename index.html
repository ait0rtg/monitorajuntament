        ❓ Guia Ràpida — Monitor Polític

        📊 Dashboard Principal

        El dashboard mostra els 5 KPIs clau, les últimes 10 novetats detectades automàticament, el gràfic de distribució per urgència i el calendari de venciments del mes.

        🔴 Sistema de Classificació

        • 🔴 URGENT — Contractació d'urgència, venciments <15 dies o imports extraordinaris
        • 🟡 IMPORTANT — PDFs JGL, imports >50.000€, temes de contractació pública
        • 🟢 INFORMATIU — Informació general sense impacte polític immediat

        🏛️ Preguntes de Ple

        La IA analitza automàticament cada document i suggereix preguntes per presentar al ple municipal. A la vista Preguntes de Ple pots copiar-les al portapapers, filtrar-les per estat i marcar-les com a presentades (canvia estat a «En Curs»).

        🔗 Connexió Supabase

        Fes clic a l'avatar R → Configuració Supabase per connectar el dashboard a la teva base de dades real. Necessites la Project URL i la Anon Key del teu projecte Supabase.

        ⚠️ En mode demostració les dades no es guarden. Configura Supabase per persistència real.

        ⌨️ Dreceres de teclat

        • ⌘K / Ctrl+K — Obrir cercador
        • Esc — Tancar modal/menú obert

        📤 Exportació de dades

        Fes clic al botó de descàrrega (↓) a la barra superior per descarregar tots els registres en format CSV compatible amb Excel (inclou BOM UTF-8, preguntes de ple i propostes d'acció).
